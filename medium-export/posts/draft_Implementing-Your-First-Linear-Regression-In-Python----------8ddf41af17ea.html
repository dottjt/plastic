What is Linear Regression?

Linear Regression is a type of supervised Machine Learning technique which aims to identify linear trends amongst a set of pre-defined features.

Features are essentially a set of random variables (Note: The term “random variable” is utterly stupid, because the variables are not actually random, instead it can help to think of the word “potential” instead of random) which can be used to predict a trend.

For example, using the size of a house (feature) in order to predict the price of a house (trend). In this case we will be performing Univariate Linear Regression, which is to say that we will only be using one feature in order to create our model/prediction.

Not to mention, Linear Regression is arguably the most basic analysis you can perform on a set of data, so it’s where I’ll be starting with this article — given I’ve literally never done this before and I actually have no idea what I’m doing, but we all have to start somewhere!
